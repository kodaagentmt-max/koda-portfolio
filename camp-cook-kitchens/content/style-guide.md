# Camp Cook Kitchens Content — Style Guide

## Voice
- Practical, hands-on, like you're cooking at camp with a buddy
- Confident tips, no fuss
- Rugged but not over-the-top
- Celebrate simple, delicious camp food

## Headlines

Use the following fixed headline for each post type:
- Daily Brief/News → 🏕️ **Outdoor Update** (NEVER use "Daily Brief")
- Camp Kitchen Hack → 🏕️ **Camp Kitchen Hack**
- Basecamp Recipe Reveal → 🏕️ **Basecamp Recipe Reveal**
- Weekly Recipe/Tip → 🔥 **Camp Cook Tip:**

Never use time-specific references like "Morning Brief," "Evening Update," or "Afternoon Special."

## Format

### Daily Brief Post (WITH image)
```
🏕️ Outdoor Update

[NEWS ITEM 1] — 1-2 sentences
[NEWS ITEM 2] — 1-2 sentences
[NEWS ITEM 3] — 1-2 sentences

Source: [where you found it]

Happy [day of week], KC
```

### Engagement Post (NO image)
```
[ENGAGEMENT QUESTION about camp cooking, gear, or outdoor experience]

Drop your answer below 👇

Camp Cook
```

### Recipe/Tip Post (WITH image)
```
🔥 Camp Cook Tip:

[RECIPE OR TIP NAME]

[Brief description — why it's worth trying]

Key ingredients/rig: [list if applicable]

Give it a shot and let us know how it turned out 👇

Camp Cook
```

## Content Rules
- Daily Brief: 2-3 news items (hunt/fish/camp), no location mentions, includes image
- Engagement: question ONLY, no image, conversational
- Recipe/Tip: practical camp cooking, includes image
- Always end with "Camp Cook"
- Do NOT repeat posts from the posted/ folder (check there first)
- Rotate images from photo-library.json — never reuse the same photo consecutively

## Seasonal Buckets

### Spring (March–May)
- Turkey season ramp-up
- Spring fishing opener
- Camp cooking: lighter meals, fresh game
- Trail mix: spring camping gear tips

### Summer (June–August)
- Bass fishing peak
- Shoreline camping
- Camp cooking: no-cook options, cold camp meals
- Grilling season, smoked meats

### Fall (Sept–Nov)
- Big game hunting season
- Migratory birds (ducks, geese)
- Camp cooking: hearty stews, Dutch oven bakes
- Campfires, comfort food, longer sessions

### Winter (Dec–Feb)
- Ice fishing, winter camping
- Camp cooking: one-pot meals, cold weather comfort
- Gear: keeping food warm, Dutch oven hacks
- Slow cooking, big batches
